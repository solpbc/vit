# lab/

Stashed experimental scripts. These are not part of the vit CLI.

## Scripts

- `plc-register.js` — Generate and register a DID:PLC genesis operation
- `plc-verify.js` — Verify a DID document and audit log from the PLC directory

## Usage

```bash
bun lab/plc-register.js --help
bun lab/plc-verify.js --help
```
